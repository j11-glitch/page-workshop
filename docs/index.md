# Spring Framework Workshop

Welcome! This workshop walks you through core Spring Boot concepts with hands-on exercises.

## What you'll learn

- How Spring Boot auto-configuration works
- Dependency Injection in practice
- Building a simple REST API

## Prerequisites

- Java 17+
- Maven 3.8+
- An IDE (IntelliJ recommended)

## Getting started

Clone the starter project:

```bash
git clone https://github.com/your-org/spring-workshop-starter.git
cd spring-workshop-starter
```

Then pick an exercise from the menu on the left.
